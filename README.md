# vuebackend
Vue‘s Backend and Frontend Personal  
If you find it useful, please start ~
```
    webpack
      |---dist 
      |---public 公共文件
        |---mock 静态模拟数据
      |---src
        |---api 数据接口
        |---assets 资源
          |---css
          |---fonts
          |---images
          |---less
        |---common 公共模块目录
        |---components组件
        |---mock动态数据模拟接口
          |---data 模拟数据
        |---views各个模块
          |---index    index模块
            |---pages 组件
              |---users 用户模块
            |---app.js
            |---IndexPage.vue
            |---index.html 页面模板
          |---about       about模块
            |---pages 组件
            |---app.js
            |---AboutPage.vue
            |---index.html 页面模板
          |---error       error404模块
            |---index.vue
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
