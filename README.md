# vuebackend
Vue‘s Backend Personal
```
    webpack
      |---dist 
      |---public 公共文件
        |---mock 模拟数据
      |---src
        |---assets 资源
        |---common 公共模块目录
        |---components组件
        |---views各个模块
          |---index    index模块
            |---pages 组件
            |---app.js
            |---IndexPage.vue
            |---index.html 页面模板
          |---about       about模块
            |---pages 组件
            |---app.js
            |---AboutPage.vue
            |---index.html 页面模板
          |---error       error404模块
            |---index.vue
```
## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```
